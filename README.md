# 需求原型演示项目

用于快速制作和展示产品需求原型的HTML页面集合。

## 项目结构

```
prototype_demos/
├── README.md           # 项目说明
├── index.html          # 项目首页（原型列表）
├── assets/             # 公共资源
│   ├── css/           # 样式文件
│   ├── js/            # 脚本文件
│   └── images/        # 图片资源
├── templates/          # 页面模板
└── demos/             # 具体原型页面
    └── demo1/         # 示例原型
```

## 技术栈

- HTML5 + CSS3 + JavaScript
- Bootstrap 5 (响应式布局)
- Font Awesome (图标)
- Vue 3 CDN (可选，用于复杂交互)

## 使用方法

1. 直接用浏览器打开 `index.html` 查看原型列表
2. 在 `demos/` 目录下创建新的原型文件夹
3. 复制 `templates/` 中的模板开始快速开发

## 特点

- 零配置，开箱即用
- 快速迭代，即时预览
- 轻量级，易于分享
- 跨平台兼容
